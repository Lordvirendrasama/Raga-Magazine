# **App Name**: RagaMagazine Frontend

## Core Features:

- Header/NavBar: Display the main navigation with links to different categories.
- HeroCarousel: Display featured posts in a carousel with autoplay, swipe, and dot indicators.
- ArticleCard Variants: Display articles in different sizes for various sections.
- MagazineGrid: Display posts tagged as magazine issues in a masonry layout with infinite scroll.
- SidebarTopStories: Display trending posts in the sidebar.
- SearchOverlay: Overlay for live search functionality hitting the WordPress REST API search endpoint as the user types.
- Netlify Configuration: Netlify configuration file including the build command and publish directory. Also, includes necessary plugins.

## Style Guidelines:

- Primary color: Black (#000000) for a bold and modern feel.
- Background color: Dark gray (#1A1A1A), a desaturated near-black that provides contrast.
- Accent color: Deep Orange (#FF6B00) to highlight key elements and CTAs.
- Headings font: 'Oswald' sans-serif for big and bold headlines. Use modern font pairings to make it feel 2025.
- Body font: 'Inter' sans-serif for a clean and readable text. Optimize for readability on large screens.
- Use simple, minimalist icons for navigation and social links. Consider using a custom icon set for a unique touch.
- Subtle animations for transitions and hover effects using Framer Motion. Implement micro-interactions for a modern feel.
- Utilize a modern and flexible grid system for a dynamic layout. Ensure responsiveness across all devices and screen sizes.
- Incorporate white space strategically to create a clean and airy design. Prioritize content hierarchy and visual balance.
